Color Changer

live github
build with: VS Code HTML CSS JavaScript

This project is a color changer task inspired by a challenge we solved during bootcamp. It involves HTML, CSS, JavaScript, and integration with an open-source API. The goal of this project is to design a webpage, replicate the design, integrate an online API to generate random colors, and provide buttons for users to view the color in different color codes.

  • Users can generate random colors by clicking a button
  • Colors are displayed on the webpage along with their RGB, HEX, and HSL color codes
  • Buttons allow users to view the color in different color code formats
